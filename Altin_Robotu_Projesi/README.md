# Gold Quant Desk v2.0

AI-Powered Technical, Fundamental, and Quantitative Analysis Desk for Gold (Gram Altın).

## Features
- **10-Layer Quant Architecture**
- **Glassmorphism UI** (Premium Design)
- **Mathematical Models**: GBM, Ito Calculus, Kelly Criterion.
- **AI Models**: LSTM, Sentiment Analysis.
- **Risk Metrics**: Sharpe, Sortino, VaR, MaxDD.

## Quick Start
```bash
pip install -r requirements.txt
python main_app.py
```
Open `http://localhost:5000` in your browser.
